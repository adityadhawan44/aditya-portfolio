# Aditya Portfolio

A high-impact personal portfolio built with React and Vite.

## Highlights
- Hero section with strong positioning
- Proof, metrics, and experience sections
- Project storytelling with problem, solution, stack, and result
- Content brand, lessons, and current-building sections
- Mobile-friendly responsive design

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
